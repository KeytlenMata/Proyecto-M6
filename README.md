# Proyecto Módulo 6 – Grupo A

## Descripción

Programa colaborativo en Python que implementa un juego de adivinación del número secreto con niveles de dificultad, límite de intentos y opción para volver a jugar.

El programa genera un número aleatorio dentro de un rango según la dificultad elegida. El jugador tiene **5 intentos** para adivinarlo, recibiendo pistas ("más alto" o "más bajo") tras cada intento. Al finalizar, se le pregunta si desea jugar nuevamente.

Este proyecto fue desarrollado por 5 integrantes del Grupo A como parte de la actividad del módulo 6.

## Características

- ✅ **Niveles de dificultad**:
  - Fácil: número entre 1 y 10
  - Medio: número entre 1 y 20
  - Difícil: número entre 1 y 50
- ✅ Máximo de **5 intentos** por partida
- ✅ Mensajes claros de retroalimentación
- ✅ Opción de **jugar de nuevo** al terminar
- ✅ Interfaz visual con separadores y emojis
