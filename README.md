# Proyecto Módulo 6 – Grupo A

## Descripción

Programa colaborativo en Python que implementa un juego de adivinación del número secreto con niveles de dificultad, límite de intentos y menú interactivo.

El programa genera un número aleatorio dentro de un rango según la dificultad elegida. El jugador tiene **5 intentos** para adivinarlo, recibiendo pistas ("el número es mayor" o "el número es menor") tras cada intento. Al finalizar una partida, se regresa automáticamente al menú principal, donde se puede jugar de nuevo, ver instrucciones o salir.

Este proyecto fue desarrollado por 5 integrantes del Grupo A como parte de la actividad del módulo 6.

## Características

- **Niveles de dificultad**:
  - Fácil: número entre 1 y 10  
  - Medio: número entre 1 y 20  
  - Difícil: número entre 1 y 50  
- Máximo de **5 intentos** por partida  
- Retroalimentación clara después de cada intento  
- Menú principal con opciones para jugar, ver instrucciones o salir  
- Interfaz visual con emojis y formato amigable
